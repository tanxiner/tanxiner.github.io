Assist Edge - Real-Time Multi-Modal Assistive Intelligence System

AssistEdge is an edge computing-based accessibility platform that enables real-time communication by integrating American Sign Language (ASL) alphabet recognition, speech-to-text, text-to-speech, and facial emotion recognition. The system assists communication between individuals who rely on sign language and the wider community by converting information between visual, audio, and text modalities.

The platform uses multiple Raspberry Pi devices to process audio and video data at the edge, providing low-latency and real-time interactions. A web-based dashboard displays recognition results and analytical insights into communication patterns and audience emotions.

Key Features

  • ASL Alphabet Recognition: Real-time recognition of American Sign Language (ASL) fingerspelling gestures and conversion into text.
  • Speech-to-Text: Converts spoken language into written text to improve accessibility for users with hearing impairments.
  • Text-to-Speech: Converts written text into spoken language to assist users with visual impairments.
  • Facial Emotion Recognition: Analyzes facial expressions to provide contextual and emotional cues during communication.
  • Edge Computing: Utilizes Raspberry Pi devices for local data processing, reducing latency and enabling real-time interactions.
  • Web-Based Dashboard: Displays real-time recognition results, transcriptions, emotion analytics, and communication insights.

Technologies Used

Language: Python, JavaScript, HTML, CSS

Frontend: React

Backend: Flask

Others: MQTT, Socket.IO, REST APIs
