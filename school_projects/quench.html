Quench E-Commerce Web Application

Quench is a simple E-commerce webpage that allows users to register, log in, edit profile, add to cart, add to wishlist, cart out, and view purchase history. Admin functionalities have also been added, such as view all profiles, and CRUD of products. Below are snippets of some of the functionalities.

Technologies Used

Language: HTML, CSS, JavaScript, PHP, SQL

Frontend: HTML, CSS, JavaScript, BootStrap

Backend: PHP, MySQL

Others: PHP Sessions, Apache/ Nginx
