BookHaven Mobile Application

BookHaven is an app developed to simplify library processes and improve the overall user experience for students. This app features biometric login, borrowing of books via QR code, view book synopsis via NFC tag, and finding of library using GPS.

Technologies Used

Language: Kotlin

Frontend: Material3, Jetpack Compose

Backend: Firebase Services, AuthRepository

Others: NFC, QR code, Google Maps, ViewModel

Biometric Login

Borrow Books via QR code Function

Book Synopsis via NFC tag Function

Find Library via GPS Function
